# Farm-Equipment-Rental-System
A full stack web application developed using Angular and Spring Boot for managing farm equipment rentals. The system allows users to register, login, add farm equipment, view available equipment, and book equipment online.
<br>
**Technologies Used**
<br>
Frontend: Angular<br>
Backend: Spring Boot<br>
Database: MySQL<br>
Language: Java, TypeScript<br>
Tools: VS Code, STS, Postman
